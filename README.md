# Boutaleb Sweet

A multilingual-ready e-commerce storefront for Boutaleb Sweet in Jeddah.

## Current MVP

- Responsive premium homepage
- Product collections and best sellers
- WhatsApp ordering via `0538007073`
- Boutaleb Sweet Jeddah branding
- Mobile, tablet and desktop layouts
- Next.js App Router with TypeScript

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
npm start
```
